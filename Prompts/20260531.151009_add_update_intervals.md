# Implementation Plan - Add 1 and 2 Seconds to Update Interval Option

We need to add options for **1 Second** and **2 Seconds** update intervals in the Settings screen of the application.

## User Interface changes (`activity_settings.xml`)
We will add two new `RadioButton` elements inside the `RadioGroup` with ID `rg_interval` in `activity_settings.xml`.

```xml
<RadioButton
    android:id="@+id/rb_1s"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="1 Second" />

<RadioButton
    android:id="@+id/rb_2s"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="2 Seconds" />
```

We will position these options before the existing 5 seconds, 10 seconds, and 15 seconds options, so that the options are sorted in ascending order of time:
1. 1 Second
2. 2 Seconds
3. 5 Seconds (Default)
4. 10 Seconds
5. 15 Seconds

## Logic changes (`SettingsActivity.java`)
We will update the `SettingsActivity` class to:
1. Check the newly added radio buttons (`rb_1s` and `rb_2s`) when loading the current setting from `SharedPreferences`.
2. Save the corresponding interval value (`1` or `2` seconds) when the radio buttons are selected.

```java
if (currentInterval == 1) ((RadioButton) findViewById(R.id.rb_1s)).setChecked(true);
else if (currentInterval == 2) ((RadioButton) findViewById(R.id.rb_2s)).setChecked(true);
else if (currentInterval == 5) ((RadioButton) findViewById(R.id.rb_5s)).setChecked(true);
...
```

```java
rgInterval.setOnCheckedChangeListener((group, checkedId) -> {
    int interval = 5;
    if (checkedId == R.id.rb_1s) interval = 1;
    else if (checkedId == R.id.rb_2s) interval = 2;
    else if (checkedId == R.id.rb_5s) interval = 5;
    ...
```
