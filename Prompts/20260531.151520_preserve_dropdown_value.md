# Implementation Plan - Preserve Selected GPX File in Dropdown during Simulation

We need to ensure that the GPX dropdown (spinner) maintains the user's selected value and is not reset, especially when the simulation starts, or when navigating to other screens (like Settings) and returning.

## Code Analysis
In [MainActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Mock_GPS_Location_APK_GITHUB/app/src/main/java/com/vypeensoft/mockgps/MainActivity.java), `scanGpxFolder()` is called in `onResume()`. When it scans the folder, it recreates the spinner adapter:
```java
ArrayAdapter<String> adapter = new ArrayAdapter<>(this, R.layout.spinner_item, fileNames);
spinnerGpx.setAdapter(adapter);
```
Setting a new adapter on `spinnerGpx` resets the selected item position to `0` (the first element). This causes the selected item to change back to the first item when the activity is paused and resumed (e.g. after returning from settings) even when the simulation is active.

## Proposed Changes

### 1. Track Service Running State
In [MockLocationService.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Mock_GPS_Location_APK_GITHUB/app/src/main/java/com/vypeensoft/mockgps/service/MockLocationService.java), we will add a static flag to expose whether the service is running. This allows the activity to immediately synchronize its UI state.

```java
private static boolean isServiceRunning = false;

public static boolean isServiceRunning() {
    return isServiceRunning;
}
```

We will set `isServiceRunning = true` in `startSimulation()` and `isServiceRunning = false` in `stopSimulation()` and `onDestroy()`.

### 2. Save Selected GPX File
In [MainActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Mock_GPS_Location_APK_GITHUB/app/src/main/java/com/vypeensoft/mockgps/MainActivity.java), we will:
- Listen to spinner selections and save the selected GPX file name in `SharedPreferences`.
- Restore the selection in `scanGpxFolder()` using the saved file name.

### 3. Sync UI State on Resume
In [MainActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Mock_GPS_Location_APK_GITHUB/app/src/main/java/com/vypeensoft/mockgps/MainActivity.java)'s `onResume()`, we will check if the service is running and update the UI state immediately, so the spinner and buttons are properly enabled/disabled without waiting for the first broadcast update.
